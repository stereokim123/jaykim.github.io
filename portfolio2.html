나만의 사진과 필터를 전시하는 모바일 전시 앱 서비스

개발기간
2024.03.14 ~ 2024.06.30

플랫폼
Android, Ios

개발인원
9명 (팀 프로젝트)

프로젝트 개요

자극적인 콘텐츠의 확산과 사용자 피로도 증가 문제를 해결하고,
깊이 있는 감상과 소통의 가치를 극대화하기 위해
모바일 사진 전시 앱 플랫폼을 개발했습니다.
사용자들은 자신만의 사진 전시회를 개최하고,
사진에 적합한 커스텀 필터를 제작할 수 있으며,
플랫폼은 커스텀 필터 수요를 충족시켜
더욱 창의적이고 의미 있는 소통을 지원하는 것을 목표로 합니다.

기여도 ★★★★★

주요 기능 및 서비스 아키텍처

1) MSA 아키텍처

SpringBoot 기반으로 User, Exhibition, Filter,
Report Service로 나눠 설계함, Apache Kafka를 사용해서
서비스간 비동기 데이터 통신 진행

2) 전시 추천 알고리즘

사용자에게 맞춤형 전시회를 추천하는 기능 제공,
Python의 Surprise Library를 사용하여
협업 필터링 기반 추천 시스템 구축

담당 역할 및 수행 업무

  • 1)User Service

    OAuth2(Google, Naver, Kakao) 기술을 활용하여 소셜로그인 구현
    Spring Security 활용하여 서비스 자체 JWT 인증/인가 구현
    Redis를 활용하여 자체 AccessToken 및 Refresh Token 관리

  • 2)Filter Service

    Custom Filter 제작 및 임시저장 CRUD API 구현
    CloudFront와 S3를 사용한 전시 업로드 구현

  • 3)Exhibition Service

    Custom Exhibition 제작 및 임시저장 CRUD API 구현
    GCP GeoCoding API 활용하여 경도/위도 값으로 주소 받아오는 로직 구현
    SQL Query문을 활용하여 지도 Clustering 및 화면 내 일정 반경 전시 정보 리스트 조회 API 구현

  • 4)Apache Kafka 통신 구현

    사용자 정보 생성, 수정, 변경, 삭제 시
    구독되어 있는 서버에서 변경내용이 처리되게끔 로직 구현
