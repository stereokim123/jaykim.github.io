프로젝트 개요

맞벌이 부부 증가와 부모 자식 간 정서적 교류 부족 문제를 해결하고,
유아의 언어발달 및 정서발달을 돕기 위해 AI 기반의 육아 보조 서비스 ‘아고미’를 개발했습니다.

아이들이 음성으로 AI 캐릭터와 상호작용하며 역할놀이를 통해
언어 능력과 감정 표현 능력을 향상시킬 수 있도록 설계되었으며,
또한 AI 분석을 통해 아이의 발달 상태를 보고서로 제공하여
부모들이 자녀의 성장을 더 깊이 이해할 수 있도록 돕는 것을 목표로 합니다.

기여도 ★★★★★

주요 기능 및 서비스 아키텍처

AI를 활용한 역할놀이 육아보조 서비스

  • LLM 파인튜닝을 통해 역할놀이에 최적화 시킨 언어모델 활용
    한국어 성능이 우수한 EEEV 모델을 기본 모델로 선정,
    커스텀 데이터셋 증강 및 파인튜닝을 통해 성능 최적화
  • 아이들의 새로운 친구인 ‘아고미’ 음성 기반 역할놀이 진행
    STT(Web Speech API)와 TTS(Google TTS API) 기술을 활용해
    ‘아고미’ 마스코트 캐릭터를 중심으로 여러 캐릭터 음성 목소리 선택하여 진행
  • 장난감 세트에 최적화된 역할놀이 상황 구성
    아이들의 집중력과 몰입도를 이끌어내기 위해 역할놀이 상황별 장난감 세트를 제공,
    장난감 세트에 맞는 데이터셋 직접 구축
  • 아이 언어발달 및 정서발달 분석
    GPT-3.5-Turbo를 활용하여 결과 분석 레포트 제공,
    부모님들이 역할놀이에 직접 참여하지 않고도
    아이들의 역할놀이를 통한 발달 정도 확인 가능

담당 역할 및 수행 업무

  • FastAPI API 구현
    FastAPI의 ORM인 SQL Alchemy 활용, 비동기 요청 처리, DB설계
  • Chat-gpt API 활용 및 Prompt Engineering
    GPT-3.5-Turbo 모델 활용, Few-shot Learning을 활용하여
    prompt engineering 수행하여 아이 발달 정도 분석
  • LangChain을 활용한 LLM EEEV 모델 서빙
    파인튜닝한 LLM EEEV 모델을 서빙하기 위해 LangChain을 활용하여
    더욱 정확하고 효율적인 응답을 제공받도록 구현
